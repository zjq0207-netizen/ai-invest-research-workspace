# AI 投资研究工作台 PRD

## 1. 产品定位

AI 投资研究工作台面向个人投资者和半专业投资者，提供信息整理、结构化分析、风险提示和研究记录沉淀。产品不直接提供个性化投资建议，而是帮助用户形成可验证、可跟踪、可复盘的投资判断。

## 2. MVP 目标

第一版聚焦四个闭环：

1. 单股分析：输入股票代码，输出投资主线、财务、估值、技术面、资金面、风险与跟踪点。
2. 新闻拆解：输入新闻，识别事件类型、影响路径、受益对象、市场定价程度和反证条件。
3. 行业链条：输入行业，展示上下游、代表公司、景气度、利润弹性和风险。
4. 横向对比：多只股票按龙头、弹性、低估、风险和适合风格排序。

## 3. 核心用户流程

### 3.1 输入 Capture

用户可输入：

- 股票代码或公司名称
- 新闻原文
- 行业关键词
- 多只股票代码

系统自动识别输入类型，并选择分析模板。如果缺少投资周期、风险偏好、市场范围等信息，系统给出追问，但同时提供默认分析。

### 3.2 分析生成

系统按以下步骤执行：

1. 识别实体和意图
2. 补齐结构化数据
3. 计算财务、估值、技术、资金指标
4. 运行评分模型
5. 生成风险与反证
6. 输出结构化报告
7. 保存报告版本和引用来源

### 3.3 研究沉淀

每次分析都会形成：

- 报告正文
- 股票卡片
- 评分明细
- 风险清单
- 逻辑失效条件
- 后续跟踪点
- 引用来源

## 4. 页面范围

### 4.1 首页 Dashboard

展示市场概览、机会雷达、风险预警、最近研究、自选跟踪和产品状态。

### 4.2 个股分析页

展示股票概览、投资评分、核心逻辑、财务质量、估值状态、技术趋势、资金面、风险与反证、跟踪计划。

### 4.3 新闻分析页

展示事件分类、影响路径、受益链条、直接/间接受益公司、市场定价程度、风险与反证。

### 4.4 研究报告页

展示完整投研报告、证据来源、模型假设、历史版本和导出入口。

### 4.5 产业链地图

展示产业链上下游节点，每个节点包含景气度、竞争格局、利润弹性、估值状态、代表公司和风险。

## 5. 评分模型

MVP 使用可解释的固定权重模型：

| 因子 | 权重 |
| --- | ---: |
| 行业景气度 | 15% |
| 产业链位置 | 10% |
| 公司竞争力 | 15% |
| 财务质量 | 15% |
| 成长确定性 | 10% |
| 估值性价比 | 15% |
| 技术趋势 | 10% |
| 资金关注度 | 5% |
| 风险惩罚 | -5% |

评级规则：

| 分数 | 评级 |
| ---: | --- |
| 85-100 | 强关注 |
| 70-84 | 关注 |
| 55-69 | 中性 |
| 40-54 | 谨慎 |
| 0-39 | 回避 |

## 6. 合规边界

产品表达统一使用：

- 可关注
- 纳入观察池
- 等待条件确认
- 降低风险暴露
- 暂不具备性价比

避免使用：

- 必买
- 必涨
- 满仓
- 保证收益
- 目标价必达

标准免责声明：

本产品提供投资研究辅助、信息整理、逻辑分析和风险提示，不构成个性化投资建议。投资决策应结合个人风险承受能力、投资期限、资产状况和专业意见独立作出。
